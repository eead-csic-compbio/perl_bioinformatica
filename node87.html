Perl5 desde Perl6

Con el módulo Inline::Perl5 es posible usar cualquier recurso de Perl5 en Perl6, como se muestra en el siguiente ejemplo con el módulo DBI: 

use v6;
use Inline::Perl5;
use DBI:from<Perl5>;

my $dbh = DBI.connect('dbi:Pg:database=test');
my $products = $dbh.selectall_arrayref(
  'select * from products', {Slice => {}}
  );
