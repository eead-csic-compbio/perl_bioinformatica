Introducción a Perl6

Aunque este curso no cubre Perl6, sí es necesario conocer lo mínimo para poder al menos ejecutar código escrito en la version 5.

Primero, como explicaba en el blog, veamos cómo instalar el intérprete de Perl6 llamado rakudo en una carpeta oculta en $HOME. Primero clonamos el código fuente con git, lo que nos permitira mas tarde actualizarlo con git pull: 

git clone https://github.com/tadzik/rakudobrew ~/.rakudobrew
echo 'export PATH=~/.rakudobrew/bin:$PATH' >> ~/.bashrc
source ~/.bashrc

Ahora compilamos tanto el interprete (en forma de moar, una máquina virtual) como el gestor de paquetes zef: 

rakudobrew build moar
rakudobrew build zef

Finalmente, instalamos el módulo Inline::Perl5. De la misma manera podemos instalar cualquiera de la lista: 

zef install Inline::Perl5

#zef --force install Inline::Perl5 # en caso de haber actualizado Perl5

Hay un tutorial sencillo para aprender Perl6 si ya sabías Perl5: 5to6-nutshell
